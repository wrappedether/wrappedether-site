# weth.io Visual Design Analysis

**Date:** 2026-01-16
**Purpose:** Analyze weth.io's visual approach to inform wrappedether.org v2.0 design

---

## Overview

weth.io was the gold standard for wrapped token education during the DeFi boom (2017-2024). This analysis examines their visual design system to extract lessons for wrappedether.org v2.0.

---

## Design System Analysis

### Color Palette

**Primary Colors:**
- **Pink/Magenta:** `#FF69B4` (hot pink) - Primary accent color
- **Black:** `#000000` - Outlines, shadows, text
- **White:** `#FFFFFF` - Clean backgrounds, coin fills
- **Light Pink:** `#FFE5E6` - Background tint (alternating sections)

**Usage Pattern:**
- Pink used strategically for emphasis (ERC-20 label, token highlights)
- High contrast (black on white, pink accents)
- Minimal color palette (3-4 colors total)
- Clean, professional, approachable

**Lesson for WETC:**
- Use ETC green (`#2ecc71`) as primary accent
- WETC blue (`#3b82f6`) for wrapped state
- Maintain high contrast for readability
- Keep palette minimal (ETC green, WETC blue, black, white, gray)

---

### Illustration Style

**Characteristics:**
- **Geometric:** Simple shapes (cubes, cylinders, jars, conveyor belts)
- **Bold outlines:** Thick black strokes (2-3px)
- **Isometric perspective:** 3D feel without complexity
- **Minimal detail:** Focus on concept, not realism
- **Playful but professional:** Friendly without being childish

**Animation Style:**
- **Smooth loops:** Continuous animations (no jarring stops)
- **Subtle motion:** Gentle movements (coins rolling, vacuum sucking)
- **Clear metaphors:** Physical actions represent digital processes
- **2-4 second loops:** Long enough to understand, short enough to not annoy

**Lesson for WETC:**
- Adopt similar geometric, isometric style
- Bold ETC green outlines
- Smooth looping animations
- Use physical metaphors (wrapping, unwrapping, transformation)

### WETH_02.gif - ERC-20 Jar with Tokens

**Visual Elements:**
- Glass jar with lid (like a cookie jar)
- Pink label: "ERC20 COMPLIANT"
- Multiple tokens inside: "TKN" (generic tokens)
- One ETH coin outside jar (can't get in)
- Light pink background

**Animation:**
- ETH coin tries to enter jar
- Bounces off (can't fit)
- Shows incompatibility

**Metaphor:**
- Jar = ERC-20 standard
- TKN tokens = compliant tokens
- ETH coin = non-compliant (can't enter DeFi)

**Educational Power:**
- Instantly shows "ETH doesn't fit"
- Visual metaphor everyone understands (cookie jar)
- Pink label draws eye to "ERC20 COMPLIANT"

**WETC Adaptation:**
- Jar labeled "ERC-20 DEFI"
- Tokens inside: "USC", "WETC", "TKN"
- ETC coin outside (can't enter)
- Animation: ETC bounces off jar
- Second state: WETC goes through (shows solution)
- Use ETC green for jar label

## Key Takeaways

### What Worked Exceptionally Well

1. **Conveyor Belt Animation (WETH_04)**
   - Clearest visualization of wrapping process
   - Industrial aesthetic builds trust
   - Perfect loop creates hypnotic effect
   - **Must adapt this for WETC**

2. **Jar Metaphor (WETH_02)**
   - Instantly understandable
   - Shows problem (ETH can't enter DeFi)
   - Universal metaphor (everyone knows cookie jars)

3. **Minimal Color Palette**
   - Pink accent stands out
   - High contrast (black/white)
   - Professional but approachable

4. **Isometric 3D Style**
   - Adds dimensionality without complexity
   - Modern, clean aesthetic
   - Works well for technical concepts

5. **Smooth Looping Animations**
   - 2-4 second loops
   - No jarring transitions
   - Subtle, pleasant to watch

### What to Improve for WETC

1. **Add Bi-Directional Animations**
   - weth.io shows wrapping only
   - WETC should show wrap AND unwrap
   - Toggle or side-by-side comparison

2. **More Explicit ETC Branding**
   - Use ETC green prominently
   - Show ETC logo on coins
   - Establish visual distinction from ETH

3. **Add Live Data Elements**
   - Total WETC supply counter
   - Number of wraps/unwraps
   - Show it's actively used

4. **Interactive Elements**
   - Click to toggle wrap/unwrap
   - Hover to see details
   - Make it engaging, not just passive

## Technical Implementation Notes

### Animation Technology

**Option 1: Animated SVG**
- Lightweight file size
- Scalable (looks sharp on retina)
- Can be styled with CSS
- Interactive (hover, click states)

**Option 2: Lottie (JSON animations)**
- After Effects workflow
- Smooth, complex animations
- Good browser support
- Larger file size than SVG

**Option 3: Framer Motion (React)**
- Component-based
- Full programmatic control
- Easy to add interactivity
- Requires JavaScript

**Recommendation:** Start with Framer Motion for v2.0
- Easier to iterate
- Full control over timing
- Can add interactivity (toggle wrap/unwrap)
- Aligns with Next.js stack

With Framer Motion + SVG, we can achieve:
- Smaller file sizes (50-100KB per animation)
- Sharper rendering (vector vs raster)
- Interactive states (hover, click)

## Conclusion

weth.io's visual approach was exceptional for wrapped token education. The conveyor belt animation (WETH_04) is the gold standard for showing wrapping visually. By adapting this style for ETC (green/blue color scheme, ETC branding, bi-directional flow), wrappedether.org v2.0 can achieve the same clarity and professionalism.
